# Diabetes Prediction ML Project
## Overview
This project predicts whether a person has diabetes using Machine Learning techniques.
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- SVM (Support Vector Machine)
- Google Colab
## Dataset
PIMA Indians Diabetes Dataset
## Features
- Data Collection and Analysis
- Data Standardization
- Train Test Split
- SVM Model Training
- Accuracy Evaluation
- Predictive System
## Accuracy
The model achieved approximately 77% accuracy on test data.
## Files Included
- Diabetes_Prediction.ipynb
- diabetes_prediction.py
- diabetes.csv
## How to Run
1. Open the project in Google Colab
2. Upload diabetes.csv dataset
3. Run all cells
4. Enter input data for prediction
## Output
The system predicts whether a person is diabetic or non-diabetic.
